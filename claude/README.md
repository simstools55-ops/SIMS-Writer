# SIMS Writer User Test RC1

このパッケージは、一般利用者テストを開始するためのClaude Project登録用ファイル一式です。

## 登録する場所

- `CLAUDE_PROJECT_INSTRUCTIONS.md`：Claude Projectの「Project Instructions」へ全文を貼り付けます。
- `knowledge/SIMS_WRITER_KNOWLEDGE_PACK.md`：Project Knowledgeへアップロードします。
- その他のファイル：手順、依頼テンプレート、実地試験記録として参照します。

最初に `SETUP_GUIDE.md`、次に `USER_TEST_REHEARSAL.md`、最後に `TEST_CHECKLIST.md` の順で進めてください。
