# Quality Framework v1.0

処理順序は Quality Audit → Evidence Verification → Consistency Audit → Quality Gate → SEO/Pattern適用 → Outputです。SEO改善を先行させません。

Quality Gate:
- proceed: 品質上の阻害要因なし
- proceed_with_warnings: 限定的な注意あり
- stop_and_rewrite: 重大な前提誤り、検索意図誤認、危険な事実誤認
- manual_review_required: 自動判断不能な高リスク事項
