# 初心者と上級者で記事品質を変えない

## 観察

User Profile別の出力差が、成果物品質差につながる可能性がある。

## SIMS Writerでの判断

記事品質は共通とし、説明・表示・根拠詳細だけを利用者別に調整する。

## active化条件

関連するKnowledge・Decision・Pattern・Quality Rule・Golden Caseとの対応を確認し、重複がないこと。
