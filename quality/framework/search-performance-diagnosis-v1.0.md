# Search Performance Diagnosis v1.0

## Position bands
- 1.0–10.0: CTR、SERP訴求、即答性を優先できる。
- 10.1–20.0: CTRと検索意図・見出し・本文充足を併用する。
- 20.1以下: CTR改善だけを主施策にせず、順位・内容・カニバリを優先する。

## Required signals
CTRだけで判断せず、平均順位、表示回数、メインクエリ順位、上位クエリ分布を併用する。

## Language policy
競合SERP等の根拠がない場合、「タイトルが主因」と断定しない。「有力な改善対象の一つ」と表現する。
