# SIMS Writer Core Principles v1.0

1. Publish Ready by Default
2. Quality by Design
3. Knowledge First
4. Evidence over Assumption
5. Contract Driven
6. Pattern over Duplication
7. Human-Centered Writing
8. Intent Complete
9. Preserve Proven Value
10. Deterministic Where Possible
11. Fail Explicitly
12. Minimum Necessary Complexity
13. Repository as Single Source of Truth
14. Version Everything
15. Test Against Real Articles
16. Learn from Operation
17. Platform Independence
18. Backward Knowledge Compatibility
19. Continuous Quality Improvement

設計判断では、安全、事実性、Publish Ready品質、検索意図、利用者の作業削減を優先する。
