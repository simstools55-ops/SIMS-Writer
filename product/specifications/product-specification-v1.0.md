# SIMS Writer Product Specification v1.0

## 対象

日本語ブログ記事の新規作成、既存記事改善、CTR改善、情報更新、部分修正、公開品質判定。

## 中核要件

- Request正規化
- Source Content Snapshot
- Knowledge Assembly
- Search Intent Modeling
- Content Planning
- Pattern Selection
- Content Production
- Quality Validation
- Targeted Refinement
- Publish Decision
- Publication Package
- Execution Record

## 必須Quality Dimensions

Safety、Factuality、Search Intent、Completeness、Helpfulness、SEO、Structure、Readability、Japanese Quality、EEAT Support、Original Value、Site Fit、Publication Readiness。

## 必須Quality Gates

Input、Planning、Evidence、Draft、SEO、Language、Final Publication。

## 初期連携

- Generic JSON Input
- SIMS-Blog-Manager Input
- Model Adapter 1種類以上

## Decision Framework Requirement Addendum

Product 1.0は変更要否、変更範囲、no_change、manual_reviewをDecision Recordとして追跡できなければならない。
