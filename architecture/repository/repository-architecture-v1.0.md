# SIMS Writer Repository Architecture v1.0

SIMS Writer Repositoryは、製品哲学、設計、Knowledge、Contract、Pattern、Quality、Runtime、Test、運用知見を一元管理するSingle Source of Truthである。

正式資産は追跡可能なID、Version、Status、根拠、依存関係を持つ。

SIMS-Coreの未評価資産は `archive/sims-core/raw/` に配置し、Keep / Adapt / Archive / Removeで分類してから現行資産へ移行する。
