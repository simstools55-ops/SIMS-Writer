# SIMS-Core Asset Mapping

| SIMS-Core資産 | SIMS Writer移行先 | 方針 |
|---|---|---|
| Project Instructions | Product / Runtime Adapter / Contract | 巨大指示を責務別に分解 |
| Engine Input Intake | Request Contract / Normalization | Engine名は廃止 |
| Intent Diagnosis | Knowledge / Decision / Planning | 判断をDecisionとして明示 |
| Article Audit | Quality Rule / Quality Report | 評価と生成を分離 |
| Patch Planner | Decision Action Plan / Improvement Pattern | 必要な変更だけを選択 |
| Rewrite Assistant | Content Production / Model Adapter | モデル交換可能化 |
| Final Check | Quality Gates | 最終確認だけに品質を依存させない |
| Management Reporter | Publication Package Contract | Requested Outputに応じて生成 |
| Knowledge資料 | Knowledge Item / Set | 主張単位に分解し出典・適用範囲を付与 |
| Templates | Contract Example / Platform Template | PatternとTemplateを分離 |
| Examples | Golden Dataset | 成功例と失敗例の双方を登録 |
